# Economic Map 프로젝트

## 개요
경제 지표 간 인과관계를 시각화하는 인터랙티브 맵
- 배포: https://machomehe.github.io/economic-map/
- GitHub: https://github.com/machomehe/economic-map

## 기술 스택
- D3.js (CDN), Vanilla JS, 단일 index.html
- 데이터: data/data.json (노드+링크), data/values.json (FRED 실시간)
- Python: fetch_data.py (FRED 데이터, 월간 통일, YoY% 변환)
- 배포: GitHub Pages (빌드 도구 없음)

## 현재 상태 (2026-04-08)
- **v5.0**: 전문가 10인 피드백 기반 대규모 개선 (Phase 0~5 완료)
- **33개 노드, 77개 관계선** (caveat 10개 포함)
- 주요 기능:
  - 가중전파 시뮬레이션 (strength/5 감쇠, depth 3, 영향도%) + 복합 시나리오
  - **대시보드** (전체 지표 신호등 그리드, 카테고리별)
  - **레짐 탐지** (PMI/금리차/VIX/NFCI 기반 → 확장/둔화/과열/위기 자동 판별)
  - **포트폴리오 영향 분석** (자산 비중 입력 → 예상 영향% + 기여도 분해)
  - 임계값 경고 배지 (VIX≥30 공포, HY≥500 스트레스, 금리차<0 침체 등)
  - 레짐 조건부 경고 (caveat: "이 관계가 깨지는 조건" 10개 링크)
  - 모든 값 변화율(%) 통일 표시
  - 레벨별 탐색 (노드 크기 차등), 줌 적응형 렌더링 (화살표/텍스트/링크 역보정)
  - 시차(lag)/인과·상관 태그, 스파크라인 (시간축+호버 툴팁)
  - 모바일 최적화 (터치, 스와이프, 더블탭줌, Safe Area, 반응형)
  - URL 파라미터, PNG 내보내기, 면책 고지, CSP 보안
- 노드: 자산5, 매크로14, 시장12, 리스크2
- 신규(v5.0): 원/달러, NFCI, Core PCE, 임금상승률, 엔/달러
- fetch_data.py: 월간 60포인트(5년) 통일, 빈도별 동적 limit, YoY% 자동 변환

## 버전 히스토리
- v4.2 (2026-04-07): Phase 1 MVP + 고객피드백 52건 중 37건 반영

  - Phase 5: 대시보드, 레짐 탐지, 포트폴리오 영향 분석
