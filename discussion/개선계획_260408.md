## Phase 0: 긴급 수정 (data.json만 수정, 1시간)

> 경제학적 오류 — 수정 안 하면 신뢰성 붕괴

### 0-1. 누락된 필수 링크 추가 (13개)

```
us_rate → us_dollar       비례  str:4  lag:즉시-1개월  causal     ← 5명 지적, 최우선
us_rate → us_gdp          반비례 str:3  lag:6-12개월   causal     ← IS곡선
us_hy → us_stocks         반비례 str:3  lag:1-3개월    correlation ← 3명 지적
us_hy → us_gdp            반비례 str:2  lag:3-6개월    causal
us_stocks → us_sentiment  비례  str:4  lag:즉시       causal     ← 부의효과
us_stocks → us_gdp        비례  str:3  lag:1-3개월    causal     ← 부의효과
us_vix → us_dollar        비례  str:3  lag:즉시       correlation ← 위기시 달러강세
us_dollar → us_stocks     반비례 str:2  lag:1-3개월    causal
us_unemployment → us_cpi  반비례 str:3  lag:6-12개월   causal     ← 필립스곡선
us_commodity → us_cpi     비례  str:2  lag:3-6개월    causal
us_realestate → us_gdp    비례  str:3  lag:3-6개월    causal
us_gdp → us_unemployment  반비례 str:4  lag:3-6개월    causal     ← 오쿤의 법칙
us_stocks → us_vix        반비례 str:4  lag:즉시       correlation
```

### 0-2. 인과방향 보완

기존 링크는 **유지**하고, 역방향 링크를 **추가**:
- `us_gdp → us_stocks` (기존) + `us_stocks → us_gdp` (신규) = 양방향
- `us_unemployment → us_gdp` (기존) + `us_gdp → us_unemployment` (신규) = 양방향
- `us_vix → us_stocks` (기존) + `us_stocks → us_vix` (신규) = 양방향

### 0-3. strength 수정 (6개)

```
us_rate → us_bonds:        5 → 4
us_cpi → us_rate:          5 → 4  (연준 재량)
us_liquidity → us_stocks:  5 → 4  (과적합)
```

### 0-4. 고립 노드 해소

## Phase 2: BFS 엔진 개선 (1-2주)

> 5명 지적. 현재 BFS의 근본적 한계 해소.

### 2-1. 가중 전파 (감쇠 적용)
```
현재: visited 체크 → 선착순 1경로만
개선: 모든 경로 탐색, 각 단계마다 strength/5 곱셈으로 감쇠
      최종 영향도 = 모든 경로의 합산
```

### 2-2. 피드백 루프 허용
- `visited` 대신 `maxDepth=3` 기반으로 루프 허용
- 2차/3차 효과를 감쇠시키며 표시
- "자기조절 메커니즘 존재" 경고 표시

### 2-3. 충격 크기 입력
```
현재: "금리가 오르면?" (이진)
개선: "금리가 [25bp / 50bp / 100bp] 오르면?" (3단계)
      strength에 크기 가중치 적용
```

### 2-4. 복합시나리오 상충 해결
- 같은 노드에 반대 방향 도달 시 strength 비교
- 순효과(net effect) 계산하여 표시

## Phase 3: 데이터 품질 (1주)

### 3-1. 빈도 통일
- 모든 데이터를 **월간** 기준으로 변환
  - 일간 → 월말값
  - 주간 → 월말값
  - 분기 → 3개월 동일값
- 스파크라인 기간: **최근 5년(60포인트)**으로 통일

### 3-2. 레짐 조건부 경고
- 레짐 민감 링크에 `caveat` 필드 추가:
  ```json
  "caveat": "인플레 레짐(CPI>4%)에서는 이 관계가 약화/반전될 수 있음"
  ```
- 바텀시트에 경고 표시

### 3-3. 임계값 경고 시스템
- VIX > 30: "시장 공포 단계"
- HY 스프레드 > 500bp: "신용 스트레스"
- 금리차 < 0: "침체 경고"
- 바텀시트에 현재값 기준 경고 배지 표시

## Phase 5: 유료 서비스화 (장기)

### 5-1. 대시보드 전환
- 각 지표 현재값 + Z-score + 신호등(빨/노/초)
- "지금 경제 상태 한눈에 보기"

### 5-2. 레짐 탐지기

### 5-3. 정량적 시뮬레이션
- rolling 회귀계수 기반 strength
- "금리 25bp↑ → S&P500 예상 -2.3%"
